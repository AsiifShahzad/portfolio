Movie Hit Prediction

A machine learning project that predicts whether a movie will be a hit or flop based on various features, helping filmmakers and producers make data-driven decisions.

Project Overview

With the growing role of data analytics in the entertainment industry, this project leverages historical movie data to train a classification model. The goal is to determine the potential success of a movie before its release, providing valuable insights to filmmakers, producers, and analysts.

Using Random Forest with extensive feature engineering and hyperparameter tuning, we've built a high-performance model that can predict a movie's success with impressive accuracy.

Key Features

Data Preprocessing

Cleaned and preprocessed movie metadata to ensure high-quality input for the model.

Feature Engineering

Advanced feature engineering on genres, production companies, and release dates.

Missing Data Handling

Sophisticated imputation techniques to handle missing data without compromising model integrity.

Hyperparameter Tuning

Optimized model performance using RandomizedSearchCV with a defined parameter grid.

Model Evaluation

Comprehensive evaluation using accuracy, confusion matrix, and classification report.

Feature Importance

Visualization of feature importance to understand key factors affecting movie success.

Machine Learning Process

1

Data Collection & Preprocessing

Gathered movie data from TMDB (The Movie Database) and performed extensive cleaning, handling missing values, and transforming categorical variables into a format suitable for machine learning.

2

Feature Engineering

Created new features from existing data, including extracting the main genre and production company, converting release dates into year and month features, and normalizing numerical values for better model performance.

3

Model Selection & Training

Selected Random Forest Classifier for its robustness and ability to handle complex relationships in the data. Split the dataset into training and testing sets to evaluate model performance accurately.

4

Hyperparameter Tuning

Used RandomizedSearchCV to find the optimal combination of hyperparameters, including number of estimators, maximum depth, and minimum samples split, significantly improving model performance.

5

Evaluation & Deployment

Evaluated the model using multiple metrics and analyzed feature importance. Created a user-friendly interface for inputting new movie data and receiving predictions.
